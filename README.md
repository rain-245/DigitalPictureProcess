## 功能说明

### 1. 边缘检测与锐化
- **Sobel 算子**：支持水平/垂直方向梯度计算，实现定向边缘检测与锐化
- **Laplace 算子**：标准二阶微分算子，全方向边缘提取
- **自定义锐化模板**：`[0 -1 0; -1 5 -1; 0 -1 0]`，直接实现图像整体锐化

### 2. 图像分割
- **固定阈值分割**：手动设置全局阈值，实现二值化分割
- **动态（自适应）阈值分割**：基于局部邻域计算阈值，适配光照不均图像
- **区域生长**：通过种子点与灰度阈值，实现连通区域分割
- **漫水分割（FloodFill）**：模拟漫水填充，快速分割单一连通目标

### 3. 形态学运算
- 支持对二值图像执行：腐蚀、膨胀、开运算、闭运算
- 可自定义结构元素尺寸（3×3、5×5 等），调整处理强度

### 4. 颜色空间转换
- 支持 BGR 图像向以下颜色空间转换：
  - HSV（适合颜色识别与分割）
  - XYZ（设备无关标准色度空间）
  - YCrCb（亮度/色度分离，适合图像压缩）

## 编译与运行
1.  克隆仓库到本地：
    ```bash
    git clone https://github.com/rain-245/DigitalPictureProcess.git
    cd DigitalPictureProcess
    ```
2.  用 Qt Creator 打开 `menu.pro` 项目文件
3.  配置好 OpenCV 库路径与包含目录
4.  构建并运行项目

---

## 使用说明
1.  点击「打开图像」按钮，加载本地图像文件（支持常见格式：`.jpg` `.png` `.bmp` 等）
2.  在对应功能模块中设置参数（如阈值、核大小、种子点坐标）
3.  点击「执行」按钮，查看处理后的图像效果
4.  可选择「保存图像」按钮，将处理结果导出到本地
